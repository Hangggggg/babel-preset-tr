# babel-preset-tr
Tr preset based on babel

## config

babel.config.json

```json
{
  "presets": [["@hangteam/babel-preset-tr", {}]]
}
```

## options

|     name      |          value           |           description           |
| :-----------: | :----------------------: | :-----------------------------: |
| autoInjection | Boolean（default：true） | Automatic inject 'Tr' variable |
